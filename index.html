Форматування тексту

Заголовки та абзаци

Для структурування великого тексту застосовуються заголовки h1–h6. Кожен розділ має свій рівень. В абзацах можна виділяти важливі поняття, позначати терміни курсивом, підсвічувати ключові фрази, або цитувати джерела.

«Хороша структура полегшує сприйняття інформації і навігацію сторінкою.»

Приклад коду вбудовано в тексті: const x = 42;. Також можна використати передформатований блок:

body { font-family: system-ui; line-height: 1.7; }
